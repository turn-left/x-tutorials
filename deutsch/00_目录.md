## 学习方法

### 每日学习流程（约90分钟）

```
┌─────────────────────────────────────────────────────────┐
│  晨间复习（15分钟）                                       │
│  ├── 复习前一日单词                                       │
│  └── 朗读上一章重点句子                                    │
├─────────────────────────────────────────────────────────┤
│  新课学习（45分钟）                                       │
│  ├── 阅读章节内容                                         │
│  ├── 记忆核心词汇                                         │
│  └── 理解语法规则                                         │
├─────────────────────────────────────────────────────────┤
│  实践练习（25分钟）                                       │
│  ├── 抄写重点句型                                         │
│  ├── 模仿造句练习                                         │
│  └── 听力跟读练习                                         │
├─────────────────────────────────────────────────────────┤
│  晚间巩固（15分钟）                                       │
│  ├── 默写当日生词                                         │
│  └── 复述本章主要内容                                     │
└─────────────────────────────────────────────────────────┘
```

### 学习原则

1. **循序渐进**：不要跳跃，按顺序学习
2. **及时复习**：每天开始前复习前一天内容
3. **开口朗读**：大声朗读例句，培养语感
4. **结合语境**：单词要放在句子中记忆
5. **做好笔记**：记录自己的难点和易错点
