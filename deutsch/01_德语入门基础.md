## 1.1 德语字母表

德语使用26个拉丁字母，外加4个特殊字符。

### 字母表（标准德语字母顺序）

| 字母 | 字母名称 | 发音特点 |
|:----:|---------|----------|
| A a | [aː] | 长元音，如 father 的 a |
| B b | [beː] | 与英语 B 相近 |
| C c | [tseː] | 在德语中较少单独出现 |
| D d | [deː] | 与英语 D 相近 |
| E e | [eː] | 短音 [ɛ]，长音 [eː] |
| F f | [ɛf] | 与英语 F 相近 |
| G g | [ɡeː] | 浊辅音，如 go |
| H h | [haː] | 清辅音 |
| I i | [iː] | 长元音，如 machine |
| J j | [jɔt] | 类似英语 y |
| K k | [kaː] | 与英语 K 相近 |
| L l | [ɛl] | 舌尖抵上颚 |
| M m | [ɛm] | 与英语 M 相近 |
| N n | [ɛn] | 与英语 N 相近 |
| O o | [oː] | 长元音，如 door |
| P p | [peː] | 与英语 P 相近 |
| Q q | [kuː] | 通常与 u 连用 |
| R r | [ɛʁ] | 小舌音（可用大舌音替代）|
| S s | [ɛs] | 在元音前发 [z]，其他发 [s] |
| T t | [teː] | 与英语 T 相近 |
| U u | [uː] | 长元音，如 food |
| V v | [faʊ] | 德语中外语词发 [v] |
| W w | [veː] | 唇齿音 [v] |
| X x | [ɪks] | 与英语 X 相近 |
| Y y | [ˈʏpsɪlɔn] | 仅在外来语中出现 |
| Z z | [tsɛt] | 始终发 [ts] 音 |
| Ä ä | [ɛː] | 类似英语 cat 的 a |
| Ö ö | [øː] | 圆唇元音，无对应英语音 |
| Ü ü | [yː] | 圆唇元音，无对应英语音 |
| ß | [ɛsˈt͡sɛt] | 发 s 音，如 plan 中的 s |

### 特殊字母说明

```
Ä (ä) - 发出类似"爱"的音
示例：Äpfel [ˈɛpfɫ̩] 苹果，Lärm [lɛʁm] 噪音

Ö (ö) - 口腔呈圆形，舌位如发 e
示例：schön [ʃøːn] 漂亮，Österreich [ˈøːstɐʁaɪç] 奥地利

Ü (ü) - 口腔呈圆形，舌位如发 i
示例：grün [ɡʁyːn] 绿色，müde [ˈmyːdə] 疲倦

ß - 只在小写字母后使用
示例：groß [ɡʁoːs] 大的，Fuß [fuːs] 脚
```

## 1.2 元音发音规则

### 单元音一览

| 元音 | 短音 | 长音 | 例词（短） | 例词（长） |
|:----:|------|------|-----------|-----------|
| a | [a] | [aː] | Mann [man] 男人 | Name [ˈnaːmə] 名字 |
| e | [ɛ] | [eː] | nett [nɛt] 友好的 | See [zeː] 湖 |
| i | [ɪ] | [iː] | bitte [ˈbɪtə] 请 | Liebe [ˈliːbə] 爱 |
| o | [ɔ] | [oː] | Gott [ɡɔt] 上帝 | Boot [boːt] 船 |
| u | [ʊ] | [uː] | Mund [mʊnt] 嘴 | Buch [buːx] 书 |

### 变元音（Umlaut）

| 变元音 | 音标 | 发音要领 | 例词 |
|--------|------|----------|------|
| ä | [ɛ] | 同 short e | Mädchen [ˈmɛːtçən] 女孩 |
| ö | [ø] | e + 圆唇 | schön [ʃøːn] 漂亮 |
| ü | [y] | i + 圆唇 | über [ˈyːbɐ] 关于 |

### 发音规则

```
┌─────────────────────────────────────────────────────────┐
│  如何判断元音长短？                                       │
├─────────────────────────────────────────────────────────┤
│  1. 相同字母双写 → 长音                                   │
│     示例：Saat [zaːt] 播种（aa = 长音）                  │
│           Boot [boːt] 船（oo = 长音）                    │
│                                                         │
│  2. 元音 + h → 长音（h 不发音）                          │
│     示例：Kahn [kaːn] 船（ah = 长音）                    │
│           Uhr [uːɐ̯] 钟表（uh = 长音）                    │
│                                                         │
│  3. 元音后无辅音或单个辅音 → 通常长音                     │
│     示例：da [daː] 那里（长音）                          │
│           hat [hat] 有（短音）                           │
│                                                         │
│  4. 词干元音后有两个以上辅音 → 短音                       │
│     示例：Mund [mʊnt] 嘴（und 前有 nt → 短音）          │
└─────────────────────────────────────────────────────────┘
```

### 组合元音

| 组合 | 发音 | 例词 | 中文 |
|------|------|------|------|
| ei | [aɪ] | mein [maɪn] | 我的 |
| ai | [aɪ] | Mai [maɪ] | 五月 |
| au | [aʊ] | Haus [haʊs] | 房子 |
| eu | [ɔʏ] | neu [nɔʏ] | 新的 |
| äu | [ɔʏ] | Häuser [ˈhɔʏsɐ] | 房子（复数） |
| ie | [iː] | Liebe [ˈliːbə] | 爱 |

### 重要辅音对比

| 辅音 | 发音 | 与英语区别 | 例词 |
|------|------|-----------|------|
| w | [v] | 类似 v 但咬唇 | Wasser [ˈvasɐ] 水 |
| v | [f] | 通常发 f 音 | Vater [ˈfaːtɐ] 父亲 |
| z | [ts] | 始终发 ts | Zeit [tsaɪt] 时间 |
| c | [k] 或 [ts] | 依后面字母 | Café [kaˈfeː] |
| s | [z] 或 [s] | 在元音间/前发 z | See [zeː] 海 / Haus [haʊs] 房子 |
| ch | [ç] 或 [x] | 软腭音 | ich [ɪç] 我 / Bach [bax] 小溪 |
| sch | [ʃ] | 类似 sh | Schule [ˈʃuːlə] 学校 |
| sp | [ʃp] | 词首发 shp | sprechen [ˈʃpʁɛçn̩] 说 |
| st | [ʃt] | 词首发 sht | Straße [ˈʃtʁaːsə] 街道 |
| t | [t] | 类似 t | Tag [taːk] 日子 |
| th | [t] | t 的发音 | Thema [ˈteːma] 主题 |
| ph | [f] | f 的发音 | Telefon [ˈteːləfoːn] 电话 |
| qu | [kv] | kw 音 | Quelle [ˈkvɛlə] 来源 |
| ck | [k] | k 的发音 | Zacke [ˈtsakə] 尖端 |
| ng | [ŋ] | 类似 thing | lang [laŋ] 长 |
| nk | [ŋk] | ng + k | Danke [ˈdaŋkə] 谢谢 |
| pf | [pf] | p + f | Pfund [pfʊnt] 磅 |
| tsch | [tʃ] | 类似 chin | Deutsch [dɔʏtʃ] 德语 |

### 清浊辅音对应

```
p - b：Paket [paˈkeːt] 包裹 - Baby [ˈbeːbi] 婴儿
t - d：Tag [taːk] 日子 - Dank [daŋk] 感谢
k - g：Karte [ˈkaʁtə] 卡片 - Garage [ɡaˈʁaːʒə] 车库
f - v：fett [fɛt] 胖的 - Vetter [ˈfɛtɐ] 堂/表兄弟
s - z：Wasser [ˈvasɐ] 水 - Zahn [tsaːn] 牙齿

重要：德语中清辅音不送气，浊辅音声带振动
```

## 1.4 德语词性系统

德语名词有三种词性：阳性（maskulin）、阴性（feminin）、中性（neutral）。

### 词性标记

| 词性 | 定冠词 | 不定冠词 | 例词 |
|------|--------|----------|------|
| 阳性 | der | ein | der Mann 男人 |
| 阴性 | die | eine | die Frau 女人 |
| 中性 | das | ein | das Kind 孩子 |

### 词性记忆规则

#### 阳性特征（der）

```
┌─────────────────────────────────────────────────────────┐
│  阳性名词通常具有以下特征：                                │
│                                                         │
│  1. 男性：der Mann, der Vater, der Sohn, der Bruder     │
│                                                         │
│  2. 以 -er 结尾：der Lehrer 老师, der Arzt 医生        │
│                                                         │
│  3. 以 -ig, -ling, -ner 结尾：                          │
│     der Honig 蜂蜜, der Schmetterling 蝴蝶             │
│                                                         │
│  4. 方向、风向、季节、月份：                              │
│     der Wind 风, der Sommer 夏天, der Januar 一月       │
│                                                         │
│  5. 以 -mus 结尾：der Tourismus 旅游                     │
│                                                         │
│  注意：这些是规律，不是绝对规则，需配合记忆                │
└─────────────────────────────────────────────────────────┘
```

#### 阴性特征（die）

```
┌─────────────────────────────────────────────────────────┐
│  阴性名词通常具有以下特征：                                │
│                                                         │
│  1. 女性：die Frau, die Mutter, die Tochter            │
│                                                         │
│  2. 以 -e 结尾（多数）：die Lampe 灯, die Tasse 杯子     │
│                                                         │
│  3. 以 -ung, -heit, -keit, -schaft 结尾：               │
│     die Wohnung 公寓, die Freiheit 自由                │
│                                                         │
│  4. 以 -tion, -sion, -age 结尾：                        │
│     die Nation 国家, die Division 部                   │
│                                                         │
│  5. 以 -ur, -ität 结尾：                                │
│     die Natur 自然, die Universität 大学                │
│                                                         │
│  复数形式通常加 -n 或 -en                                │
└─────────────────────────────────────────────────────────┘
```

#### 中性特征（das）

```
┌─────────────────────────────────────────────────────────┐
│  中性名词通常具有以下特征：                                │
│                                                         │
│  1. 以 -chen, -lein, -el 结尾（表示小称）：               │
│     das Mädchen 女孩, das Büchlein 小书                 │
│                                                         │
│  2. 动词不定式名词化：                                    │
│     das Essen 食物, das Trinken 喝                      │
│                                                         │
│  3. 颜色、金属、国家大陆名：                              │
│     das Rot 红色, das Gold 金, das Auto 汽车           │
│                                                         │
│  4. 以 -ment, -tum 结尾：                               │
│     das Instrument 乐器, das Eigentum 财产              │
│                                                         │
│  5. 动名词（das + 动词词干）：                           │
│     das Lesen 阅读, das Schreiben 写作                   │
└─────────────────────────────────────────────────────────┘
```

## 1.5 定冠词与不定冠词

### 冠词变化表

|  | 阳性 | 阴性 | 中性 | 复数 |
|:--:|:----:|:----:|:----:|:----:|
| 定冠词 | der | die | das | die |
| 不定冠词 | ein | eine | ein | - |

### 使用规则

```
┌─────────────────────────────────────────────────────────┐
│  定冠词 der/die/das 的使用场景：                         │
│                                                         │
│  1. 再次提到的事物                                       │
│     Ich habe einen Hund. Der Hund ist brav.            │
│     (我有一只狗。这只狗很乖。)                            │
│                                                         │
│  2. 独一无二的事物                                       │
│     Die Sonne scheint. (太阳照耀。)                      │
│     Der Mond ist hell. (月亮很亮。)                     │
│                                                         │
│  3. 整体概念                                             │
│     Der Kaffee ist in Deutschland sehr beliebt.       │
│     (咖啡在德国很受欢迎。)                               │
│                                                         │
│  4. 最高级修饰的名词                                     │
│     Das ist das beste Restaurant.                      │
│     (这是最好的餐厅。)                                   │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  不定冠词 ein/eine/ein 的使用场景：                      │
│                                                         │
│  1. 首次提到的事物                                       │
│     Ich möchte einen Kaffee. (我想喝杯咖啡。)            │
│                                                         │
│  2. 职业、身份                                           │
│     Er ist ein Arzt. (他是医生。)                       │
│                                                         │
│  3. 表示"一个/某个"                                      │
│     Ein Student fragt. (一个学生提问。)                  │
└─────────────────────────────────────────────────────────┘
```

### 否定形式

| 肯定形式 | 否定形式 |
|----------|----------|
| ein | kein / keine |
| eine | keine |
| ein | kein |

```
示例：
Ich habe ein Auto. → Ich habe kein Auto.
(我有一辆车。)       (我没有车。)

Ich habe eine Katze. → Ich habe keine Katze.
(我有一只猫。)       (我没有猫。)
```

## 1.6 德语基本语序

### 陈述句语序

**标准语序：主语 + 动词 + 其他成分**

```
Ich lerne Deutsch.          (我学习德语)
Ich trinke Kaffee.          (我喝咖啡)
Er wohnt in Berlin.         (他住在柏林)
Sie arbeitet bei Siemens.   (她在西门子工作)
```

**动词在第二位**：无论主语长短，动词始终在第二位。

```
┌─────────────────────────────────────────────────────────┐
│  正常语序                                               │
│  Ich        lerne    Deutsch.                           │
│  主语(第一位)  动词      宾语                            │
│                                                         │
│  如果有时间状语呢？                                       │
│  Ich        lerne    heute   Deutsch.                  │
│  主语        动词     状语      宾语                     │
│                                                         │
│  状语提前呢？                                             │
│  Heute      lerne    ich     Deutsch.                  │
│  状语(第一位) 动词      主语     宾语                    │
│                                                         │
│  注意：动词依然在第二位！                                 │
└─────────────────────────────────────────────────────────┘
```

### 一般疑问句语序

**动词提前到第一位**

```
Lernst du Deutsch?           (你学习德语吗？)
Trinkst du Kaffee?          (你喝咖啡吗？)
Kommt er aus China?         (他来自中国吗？)
```

### 特殊疑问句语序

**疑问词 + 动词 + 主语 + 其他**

| 疑问词 | 含义 | 例句 |
|--------|------|------|
| Wer | 谁 | Wer bist du? (你是谁？) |
| Was | 什么 | Was machst du? (你做什么？) |
| Wo | 在哪里 | Wo wohnst du? (你住哪里？) |
| Woher | 从哪里 | Woher kommst du? (你从哪里来？) |
| Wohin |去哪里 | Wohin gehst du? (你去哪里？) |
| Wann | 什么时候 | Wann kommst du? (你什么时候来？) |
| Warum / Wieso | 为什么 | Warum lernst du Deutsch? (为什么学德语？) |
| Wie | 怎么样 | Wie geht es dir? (你好吗？) |

### 连词开头的句子

```
┌─────────────────────────────────────────────────────────┐
│  常见连词：und, aber, oder, denn, sondern, weil         │
│                                                         │
│  规则：连词开头的句子，动词仍在第二位                      │
│                                                         │
│  Ich bin müde, denn ich habe lange gearbeitet.          │
│  (我累了，因为我工作了很久。)                              │
│                                                         │
│  Er lernt Deutsch, aber sie lernt Spanisch.            │
│  (他学德语，但她学西班牙语。)                              │
└─────────────────────────────────────────────────────────┘
```

## 1.7 音节划分规则

### 基本原则

```
1. 元音不能分开
   le-ren (正确)  /  l-er-en (错误)

2. 辅音组合一般不分开
   Stra-ße (正确)  /  S-tra-ße (错误)

3. 两个元音之间分开
   schrei-ben (正确)

4. 前缀分开（be-, ge-, er-, ver-, zer-, ent-）
   ge-lernt, be-kom-men, Er-de (地球)
```

### 常见前缀

| 前缀 | 说明 | 例词 |
|------|------|------|
| be- | 使... | beginnen 开始 |
| ge- | 完成/过去 | gemacht 做了 |
| er- | 完成 | erklären 解释 |
| ver- | 变化/完成 | verstehen 理解 |
| zer- | 破碎 | zerbrechen 打碎 |
| ent- | 开始/去除 | entstehen 产生 |
