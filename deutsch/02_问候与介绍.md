## 2.1 基本问候语

### 按时间分类

| 时间段 | 德语 | 中文 | 使用场景 |
|--------|------|------|----------|
| 全天通用 | Hallo | 你好 | 熟悉的人之间，非正式 |
| 白天通用 | Guten Tag | 您好/你好 | 正式场合最常用 |
| 早晨 | Guten Morgen | 早上好 | 上午用 |
| 下午 | Guten Nachmittag | 下午好 | 下午使用（较少见）|
| 傍晚 | Guten Abend | 晚上好 | 傍晚后使用 |
| 告别 | Auf Wiedersehen | 再见 | 正式 |
| 告别 | Tschüss / Ciao | 再见 | 非正式 |
| 晚安 | Gute Nacht | 晚安 | 睡前 |

### 详细例句

```
早上（上午好）
A: Guten Morgen, Frau Müller!
   (早上好，米勒女士！)
B: Guten Morgen, Herr Schmidt!
   (早上好，施密特先生！)

白天（您好）
A: Guten Tag! Ich bin Anna.
   (您好！我是Anna。)
B: Guten Tag, Anna. Ich bin Peter.
   (您好，Anna。我是Peter。)

傍晚（晚上好）
A: Guten Abend!
   (晚上好！)
B: Guten Abend!
   (晚上好！)

告别（正式）
A: Auf Wiedersehen! Schönen Tag noch!
   (再见！祝您愉快！)
B: Auf Wiedersehen!Likewise!
   (再见！您也是！)

告别（非正式）
A: Tschüss! Bis morgen!
   (再见！明天见！)
B: Ciao! Bis dann!
   (再见！回头见！)
```

### 特殊问候表达

| 表达 | 中文 | 使用场景 |
|------|------|----------|
| Grüß Gott! | 您好 | 南部德国（巴伐利亚等）|
| Grüß dich! | 你好 | 非常随意的问候 |
| Servus! | 你好/再见 | 奥地利、南德 |
| Moin! | 你好 | 北德地区 |
| Tag! | 你好 | 简单随意的问候 |

## 2.2 自我介绍

### 基本句型

#### 我叫...

```
Ich heiße Anna.           (我叫Anna。)
Mein Name ist Anna.       (我的名字是Anna。)
Ich bin Anna.             (我是Anna。)
```

#### 我来自...

```
Ich komme aus China.      (我来自中国。)
Ich komme aus Beijing.    (我来自北京。)
Ich komme von China.      (我来自中国。)
```

#### 我...岁

```
Ich bin 25 Jahre alt.     (我25岁了。)
Ich bin 30.               (我30岁。)
```

#### 我是...

```
Ich bin Student.          (我是学生。)
Ich bin Lehrerin.          (我是老师。)
Ich bin Arzt.             (我是医生。)
```

### 组合自我介绍

```
┌─────────────────────────────────────────────────────────┐
│  完整版                                                  │
│                                                         │
│  Guten Tag! Ich heiße Wang Lei. Ich komme aus China,   │
│  aus der Stadt Beijing. Ich bin 28 Jahre alt.          │
│  Ich bin Student und studiere Informatik.              │
│                                                         │
│  (您好！我叫王磊。我来自中国北京。我28岁。我是学生，       │
│  专业是计算机科学。)                                      │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  简化版                                                  │
│                                                         │
│  Hallo! Ich bin Anna. Ich komme aus Deutschland.       │
│                                                         │
│  (你好！我是Anna。我来自德国。)                           │
└─────────────────────────────────────────────────────────┘
```

### 正式自我介绍（面试/商务）

```
Sehr geehrte Damen und Herren,
(尊敬的女士们先生们，)

ich möchte mich kurz vorstellen.
(请允许我简单自我介绍。)

Mein Name ist Thomas Müller. Ich bin 35 Jahre alt und 
komme aus Hamburg. Ich habe fünf Jahre Erfahrung im 
Bereich Marketing.

(我叫Thomas Müller。35岁，来自汉堡。我在市场营销领域
有五年经验。)

Vielen Dank für Ihre Aufmerksamkeit.
(感谢您的关注。)
```

## 2.3 询问他人信息

### 询问姓名

| 正式程度 | 问句 | 回答 |
|----------|------|------|
| 正式 | Wie heißen Sie? | Ich heiße... |
| 非正式 | Wie heißt du? | Ich heiße... / Ich bin... |
| 非常随意 | Und du? | ... |

```
正式场合：
A: Guten Tag! Wie heißen Sie, bitte?
   (您好！请问您叫什么名字？)
B: Ich heiße Anna Schmidt.
   (我叫Anna Schmidt。)

非正式场合：
A: Hi! Wie heißt du?
   (嗨！你叫什么名字？)
B: Ich bin Peter.
   (我是Peter。)
```

### 询问国籍/来自哪里

| 问句 | 中文 | 回答方式 |
|------|------|----------|
| Woher kommen Sie? | 您来自哪里？ | Ich komme aus + 国家/城市 |
| Woher kommst du? | 你来自哪里？ | 同上 |
| Aus welchem Land kommen Sie? | 您来自哪个国家？ | Ich komme aus + 国家 |

```
A: Woher kommen Sie?
   (您来自哪里？)
B: Ich komme aus China.
   (我来自中国。)

A: Aus welchem Land kommen Sie?
   (您来自哪个国家？)
B: Ich komme aus der Volksrepublik China.
   (我来自中华人民共和国。)

补充：常见国家名
┌─────────────────────────────────────────────────────────┐
│  die Schweiz 瑞士 | Österreich 奥地利 | France 法国    │
│  Italien 意大利 | Spanien 西班牙 | England 英格兰      │
│  Amerika 美国 | Japan 日本 | Korea 韩国                 │
│  die Türkei 土耳其 | Russland 俄罗斯                   │
└─────────────────────────────────────────────────────────┘
```

### 询问年龄

| 问句 | 中文 | 回答 |
|------|------|------|
| Wie alt sind Sie? | 您多大了？ | Ich bin ... Jahre alt. |
| Wie alt bist du? | 你多大了？ | Ich bin ... |
| Wie alt ist er/sie? | 他/她多大了？ | Er/Sie ist ... |

```
A: Wie alt sind Sie?
   (您多大了？)
B: Ich bin 30 Jahre alt.
   (我30岁了。)

A: Und wie alt ist Ihre Schwester?
   (那您妹妹多大了？)
B: Sie ist 25.
   (她25岁。)
```

### 询问职业

| 问句 | 中文 | 回答 |
|------|------|------|
| Was sind Sie von Beruf? | 您职业是什么？ | Ich bin ... |
| Was machst du beruflich? | 你做什么工作？ | Ich arbeite als ... |
| Wo arbeiten Sie? | 您在哪里工作？ | Ich arbeite bei/in ... |

```
A: Was sind Sie von Beruf?
   (您是做什么工作的？)
B: Ich bin Lehrerin. Ich unterrichte Deutsch an einer 
   Universität.
   (我是老师。我在大学教德语。)

A: Und was macht Ihr Mann?
   (您丈夫做什么工作？)
B: Er ist Ingenieur. Er arbeitet bei Siemens.
   (他是工程师。他在西门子工作。)
```

### 询问健康状况

| 问句 | 中文 | 回答 |
|------|------|------|
| Wie geht es Ihnen? | 您好吗？ | Sehr gut, danke! |
| Wie geht's? | 你好吗？ | Gut, danke! |
| Wie geht's dir? | 你好吗？ | Nicht schlecht! |

```
A: Guten Tag, Herr Schmidt! Wie geht es Ihnen?
   (您好，施密特先生！您好吗？)
B: Sehr gut, danke! Und Ihnen?
   (很好，谢谢！您呢？)
A: Auch gut, danke!
   (也很好，谢谢！)

较随意：
A: Hey, wie geht's?
   (嘿，你好吗？)
B: Ganz gut, und dir?
   (挺好的，你呢？)
A: Na ja, könnte besser sein...
   (哎，还行吧...)
```

### 完整对话示例

```
A: Guten Tag! Ich bin Maria. Und wie heißen Sie?
   (您好！我是Maria。您叫什么名字？)

B: Guten Tag, Maria! Ich bin Hans.
   (您好，Maria！我是Hans。)

A: Freut mich, Hans! Woher kommen Sie?
   (很高兴认识你，Hans！您来自哪里？)

B: Ich komme aus Österreich. Und Sie?
   (我来自奥地利。您呢？)

A: Ich komme aus Deutschland, aus München.
   (我来自德国，慕尼黑。)

B: Oh, sehr schön! Wie alt sind Sie?
   (哦，真好！您多大了？)

A: Ich bin 28. Und Sie?
   (我28岁。您呢？)

B: Ich bin 30. Was sind Sie von Beruf?
   (我30岁。您是做什么工作的？)

A: Ich bin Ärztin. Und Sie?
   (我是医生。您呢？)

B: Ich bin Student. Ich studiere Musik.
   (我是学生。我学音乐。)

A: Sehr interessant! Na, dann bis später!
   (很有趣！那回头见！)

B: Ja, bis später! Auf Wiedersehen!
   (好，回头见！再见！)
```

## 2.4 正式与非正式表达

### 人称代词对照

| 人称 | 主格 | 使用场景 |
|------|------|----------|
| ich | 我 | 称呼自己 |
| du | 你 | 熟悉的人、朋友、家人、儿童 |
| er/sie/es | 他/她/它 | 第三人称 |
| wir | 我们 | 称呼包括自己在内的群体 |
| ihr | 你们 | 称呼多个熟悉的人 |
| sie | 他们/她们 | 第三人称复数 |
| Sie | 您 | 正式场合称呼对方 |

### 正式与动词形式对比

```
┌─────────────────────────────────────────────────────────┐
│  以动词 "heißen" (叫) 为例：                             │
├─────────────────────────────────────────────────────────┤
│  非正式 du        正式 Sie                               │
│  ich heiße...    ich heiße...  (相同)                   │
│  du heißt...     Sie heißen... (不同！)                 │
│  er heißt...     er/sie heißt...                       │
│  wir heißen...   wir heißen...                          │
│  ihr heißt...    Sie heißen...                          │
│  sie heißen...   sie heißen...                          │
└─────────────────────────────────────────────────────────┘
```

### Sie vs du 使用规则

```
┌─────────────────────────────────────────────────────────┐
│  使用 Sie (正式) 的场合：                                │
│  · 第一次见面的成年人                                    │
│  · 商务/职场环境                                        │
│  · 对长辈、年长者                                       │
│  · 顾客、陌生人                                         │
│                                                         │
│  使用 du (非正式) 的场合：                               │
│  · 朋友之间                                            │
│  · 家人之间                                            │
│  · 同学、同事之间（如果关系密切）                         │
│  · 对儿童                                              │
│  · 派对、聚会等轻松场合                                 │
└─────────────────────────────────────────────────────────┘
```

### 何时转换称呼

```
┌─────────────────────────────────────────────────────────┐
│  从 Sie 转到 du 的信号：                                 │
│                                                         │
│  1. 对方说："Wir können uns duzen."                     │
│     (我们可以互相用"你"称呼。)                            │
│                                                         │
│  2. 对方说："Darf ich du sagen?"                        │
│     (我可以叫你"你"吗？)                                 │
│                                                         │
│  3. 握手后说："Freut mich, dich kennenzulernen."        │
│     (很高兴认识你。)                                     │
│                                                         │
│  注意：德国人通常不会很快从Sie转到du，需要时间            │
└─────────────────────────────────────────────────────────┘
```

## 2.5 物主冠词

### 基本物主冠词（mein, dein, sein...）

| 人称 | 阳性 | 阴性 | 中性 | 复数 |
|------|------|------|------|------|
| ich | mein | meine | mein | meine |
| du | dein | deine | dein | deine |
| er | sein | seine | sein | seine |
| sie | ihr | ihre | ihr | ihre |
| es | sein | seine | sein | seine |
| wir | unser | unsere | unser | unsere |
| ihr | euer | eure | euer | eure |
| sie/Sie | ihr | ihre | ihr | ihre |

### 使用示例

```
┌─────────────────────────────────────────────────────────┐
│  阳性 (der)                                              │
│  Das ist mein Hund. Ich habe einen Hund.               │
│  (这是我的狗。我有一只狗。)                               │
│                                                         │
│  阴性 (die)                                              │
│  Das ist meine Katze. Ich habe eine Katze.            │
│  (这是我的猫。我有一只猫。)                               │
│                                                         │
│  中性 (das)                                              │
│  Das ist mein Baby. Ich habe ein Baby.                │
│  (这是我的宝宝。我有一个宝宝。)                            │
│                                                         │
│  复数 (die)                                              │
│  Das sind meine Kinder. Ich habe Kinder.              │
│  (这是我的孩子们。我有孩子。)                              │
└─────────────────────────────────────────────────────────┘
```

### 名词第四格（否定）

```
kein 的变格：
┌─────────────────────────────────────────────────────────┐
│  阳性              阴性              中性                │
│  kein Hund        keine Katze      kein Baby          │
│  Ich habe keinen Hund. Ich habe keine Katze. Ich habe kein Baby.  │
│  (我没有狗。)       (我没有猫。)       (我没有宝宝。)      │
└─────────────────────────────────────────────────────────┘
```

## 课后练习

### 练习1：配对练习

将问候语与对应的时间连接：

```
1. Guten Morgen         a. 晚上告别
2. Gute Nacht          b. 早晨问候
3. Auf Wiedersehen     c. 白天问候
4. Guten Tag           d. 晚上睡觉
5. Tschüss             e. 非正式告别
```

### 练习2：改写句子

将下列句子改为正式表达：

```
1. Wie heißt du?
   → ____________?

2. Ich bin 20 Jahre alt.
   → Ich bin 20 Jahre alt.
   (年龄无需改变)

3. Woher kommst du?
   → ____________?

4. Wie geht's?
   → ____________?
```

### 练习3：完成对话

```
A: Guten Tag! _____ Sie?
B: Ich _______ Anna.
A: Woher _______ Sie?
B: Ich _______ aus China.
A: Wie _______ sind Sie?
B: Ich bin 25.
A: Freut mich!
B: ______ auch!
```

### 练习4：用物主冠词填空

```
1. Das ist _____ (我的) Haus.
2. Das ist _____ (你的) Katze.
3. Das ist _____ (他的) Buch.
4. Das ist _____ (她的) Freund.
5. Das sind _____ (我们的) Kinder.
```

---

*答案：*
*练习1: 1-b, 2-d, 3-a, 4-c, 5-e*
*练习2: Wie heißen Sie?; Woher kommen Sie?; Wie geht es Ihnen?*
*练习3: Wie heißen; heiße; kommen; komme; alt; Freut*
*练习4: mein; deine; sein; ihr; unsere*
