## 3.2 基数词 21-100

### 构成规则

```
┌─────────────────────────────────────────────────────────┐
│  德语数字构成：逆向思维！                                │
│                                                         │
│  21 = einundzwanzig  (1 + und + 20)                    │
│     字面意思：20和1                                      │
│                                                         │
│  32 = zweiunddreißig  (2 + und + 30)                   │
│     字面意思：30和2                                      │
│                                                         │
│  45 = fünfundvierzig  (5 + und + 40)                   │
│     字面意思：40和5                                      │
│                                                         │
│  公式：个位 + und + 十位                                 │
└─────────────────────────────────────────────────────────┘
```

### 21-100 数字表

| 数字 | 德语 | 数字 | 德语 |
|:----:|------|:----:|------|
| 21 | einundzwanzig | 30 | dreißig |
| 22 | zweiundzwanzig | 40 | vierzig |
| 23 | dreiundzwanzig | 50 | fünfzig |
| 24 | vierundzwanzig | 60 | sechzig |
| 25 | fünfundzwanzig | 70 | siebzig |
| 26 | sechsundzwanzig | 80 | achtzig |
| 27 | siebenundzwanzig | 90 | neunzig |
| 28 | achtundzwanzig | 100 | (ein)hundert |
| 29 | neunundzwanzig | | |

### 十位数特殊变化

```
注意：6和7的词干变化

6: sechs → sechzig (60), sechshundert (600)
7: sieben → siebzig (70), siebenhundert (700)

其他数字无变化：
3: drei → dreißig (30), dreihundert (300)
4: vier → vierzig (40), vierhundert (400)
5: fünf → fünfzig (50), fünfhundert (500)
```

### 例句

```
Meine Telefonnummer ist dreißig, fünfzehn, zwanzig.
(我的电话号码是30-15-20。)

Das kostet fünfundvierzig Euro.
(这个45欧元。)

Er ist fünfundzwanzig Jahre alt.
(他25岁。)

Wir brauchen noch hundert.
(我们还需要一百个。)
```

## 3.4 序数词

### 构成规则

```
┌─────────────────────────────────────────────────────────┐
│  序数词构成：                                            │
│                                                         │
│  1. 基本规则：基数词 + t                                 │
│     4 → vierte, 5 → fünfte, 6 → sechste                │
│                                                         │
│  2. 特殊变化：                                          │
│     1 → erste (不规则)                                  │
│     2 → zweite (不规则)                                  │
│     3 → dritte (不规则)                                  │
│     7 → siebte (去掉 -en)                               │
│     8 → achte (不变)                                    │
│                                                         │
│  3. 10以上：基数词 + st                                 │
│     10 → zehnte → zehnte (10te错误!)                   │
│     11 → elfte, 12 → zwölfte                           │
│                                                         │
│  4. 词尾：配合定冠词 der/die/das 使用                    │
│     der erste, die zweite, das dritte                 │
└─────────────────────────────────────────────────────────┘
```

## 3.5 时间表达

### 整点表达

```
┌─────────────────────────────────────────────────────────┐
│  整点句型：                                              │
│  Es ist ___ Uhr. (现在是___点。)                         │
│                                                         │
│  1:00  Es ist ein Uhr. / Es ist eins.                  │
│  2:00  Es ist zwei Uhr.                                 │
│  5:00  Es ist fünf Uhr.                                │
│  12:00 Es ist zwölf Uhr.                               │
│  24:00 Mitternacht. (午夜)                              │
│                                                         │
│  注意：德语中 1点 用 "ein Uhr"，不说 "eins Uhr"          │
└─────────────────────────────────────────────────────────┘
```

### 半点以前

```
┌─────────────────────────────────────────────────────────┐
│  半点及半点以前：                                        │
│                                                         │
│  1:15  Viertel nach eins / ein Uhr fünfzehn            │
│  1:30  halb zwei                                       │
│  1:45  Viertel vor zwei / drei Viertel zwei            │
│                                                         │
│  重要规则：                                              │
│  ┌─────────────────────────────────────────────────┐    │
│  │ 德语时间以"半小时"为分界点！                       │    │
│  │                                                   │    │
│  │ 1:30 = halb ZWEI（半点后算下一个小时）             │    │
│  │ 1:45 = Viertel vor ZWEI（45分后算下一个小时）     │    │
│  │                                                   │    │
│  │ 也就是说：halb zwei = 1:30，不是1:30！           │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

### 半点以后

```
┌─────────────────────────────────────────────────────────┐
│  半点及半点以后：                                        │
│                                                         │
│  2:15  Viertel nach zwei                               │
│  2:30  halb drei                                      │
│  2:45  Viertel vor drei                               │
│                                                         │
│  对照记忆：                                              │
│  ┌─────────────────────────────────────────────────┐    │
│  │ 1:30 = halb ZWEI        2:30 = halb DREI        │    │
│  │ 1:45 = Viertel vor ZWEI 2:45 = Viertel vor DREI│    │
│  │                                                   │    │
│  │ 记忆技巧：halb 和 vor 都是"看下一个小时"           │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

### 分钟精确表达

```
7:05  sieben Uhr fünf
7:10  sieben Uhr zehn
7:12  sieben Uhr zwölf
7:20  sieben Uhr zwanzig
7:25  sieben Uhr fünfundzwanzig
7:35  sieben Uhr fünfunddreißig
7:40  sieben Uhr vierzig
7:50  sieben Uhr fünfzig
7:58  sieben Uhr achtundfünfzig
```

### 询问时间

| 问句 | 中文 |
|------|------|
| Wie spät ist es? | 现在几点了？ |
| Wie viel Uhr ist es? | 现在几点了？ |
| Können Sie mir sagen, wie spät es ist? | 您能告诉我现在几点吗？ |
| Hast du Uhr? | 你带表了吗？（口语） |

### 12小时制与24小时制

```
┌─────────────────────────────────────────────────────────┐
│  德国日常使用12小时制，但官方/交通用24小时制              │
│                                                         │
│  12小时制（日常）：                                       │
│  Morgens:  7:00 morgens / 7:00 früh                    │
│  Abends:   7:00 abends                                 │
│  Nachts:   7:00 nachts                                 │
│                                                         │
│  24小时制（火车、飞机、广播）：                            │
│  07:00 Uhr, 13:00 Uhr, 19:00 Uhr                       │
│                                                         │
│  常用时间短语：                                          │
│  · morgens 早上      · mittags 中午                     │
│  · nachmittags 下午  · abends 晚上                      │
│  · nachts 夜里       · heute Morgen 今天早上            │
│  · morgen früh 明天早上                                 │
└─────────────────────────────────────────────────────────┘
```

### 时间对话示例

```
A: Entschuldigung, wie spät ist es?
   (打扰一下，现在几点了？)

B: Es ist halb drei.
   (现在两点半。)

A: Danke schön!
   (非常感谢！)

---

A: Wann fährt der Zug nach Berlin?
   (去柏林的火车什么时候开？)

B: Um vierzehn Uhr dreißig.
   (14点30分。)

A: Und wann kommt er in Berlin an?
   (什么时候到达柏林？)

B: Um siebzehn Uhr fünfzig.
   (17点50分。)
```

### 周末两天说法

```
┌─────────────────────────────────────────────────────────┐
│  在德国，周末指：                                        │
│  · Samstag (周六)                                       │
│  · Sonntag (周日)                                       │
│                                                         │
│  也可以说：                                             │
│  · das Wochenende (周末)                                │

```

### 星期用法

```
┌─────────────────────────────────────────────────────────┐
│  介词 am + 星期                                          │
│  am Montag (在周一)                                     │
│  am Dienstag (在周二)                                   │
│  ...                                                    │
│                                                         │
│  每个星期...：jeden + 星期                              │
│  jeden Montag (每周一)                                  │
│  jeden Mittwoch (每周三)                                │
│                                                         │
│  在周末...：am Wochenende                               │
│  Am Wochenende arbeite ich nicht.                       │
│  (周末我不工作。)                                        │
└─────────────────────────────────────────────────────────┘
```

### 星期对话

```
A: Welcher Tag ist heute?
   (今天星期几？)

B: Heute ist Montag.
   (今天周一。)

A: Und morgen?
   (那明天呢？)

B: Morgen ist Dienstag.
   (明天周二。)

A: Haben Sie am Mittwoch Zeit?
   (您周三有空吗？)

B: Ja, am Mittwoch bin ich frei.
   (是的，周三我有空。)
```
