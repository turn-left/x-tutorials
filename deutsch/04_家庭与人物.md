## 4.3 描述家庭

### 基本句型

```
Meine Familie ist klein / groß.
(我的家庭小/大。)

Ich habe eine große Familie.
(我有一个大家庭。)

Wie viele Geschwister hast du?
(你有几个兄弟姐妹？)

Ich habe keine Geschwister. Ich bin Einzelkind.
(我没有兄弟姐妹。我是独生子/女。)

Hast du Geschwister?
(你有兄弟姐妹吗？)
Ja, ich habe einen Bruder und eine Schwester.
(是的，我有一个兄弟和一个姐妹。)
```

### 介绍家庭成员

```
Das ist meine Familie.
(这是我的家人。)

Das ist mein Vater. Er heißt Peter.
(这是我父亲。他叫Peter。)

Meine Mutter ist 55 Jahre alt.
(我母亲55岁了。)

Meine Eltern leben in Berlin.
(我父母住在柏林。)

Ich habe zwei Brüder. Der ältere heißt Thomas, 
der jüngere heißt Michael.
(我有两个兄弟。大的叫Thomas，小的叫Michael。)
```

### 家庭关系表达

```
┌─────────────────────────────────────────────────────────┐
│  家庭成员关系：                                          │
│                                                         │
│  der ältere Bruder = 哥哥                               │
│  der jüngere Bruder = 弟弟                             │
│  die ältere Schwester = 姐姐                            │
│  die jüngere Schwester = 妹妹                          │
│                                                         │
│  祖辈关系：                                              │
│  die Großeltern = 祖父母们                              │
│  die Urgroßeltern = 曾祖父母                            │
│                                                         │
│  配偶关系：                                              │
│  verheiratet = 已婚                                     │
│  ledig = 未婚                                           │
│  geschieden = 离婚                                      │
│  verwitwet = 丧偶                                       │
└─────────────────────────────────────────────────────────┘
```

### 询问家庭

```
A: Hast du Geschwister?
   (你有兄弟姐妹吗？)

B: Ja, ich habe eine Schwester.
   (是的，我有一个姐妹。)

A: Wie alt ist sie?
   (她多大了？)

B: Sie ist 22 Jahre alt.
   (她22岁了。)

A: Was macht sie?
   (她是做什么的？)

B: Sie studiert Medizin.
   (她学医。)
```

### 描述家庭住址

```
Wo wohnst du?
(你住哪里？)

Ich wohne bei meinen Eltern.
(我住在父母家。)

Ich wohne noch bei meinen Eltern.
(我还住在父母家。)

Meine Eltern wohnen in München.
(我父母住在慕尼黑。)

Ich ziehe nächste Woche um.
(我下周搬家。)

Ich suche eine Wohnung in Berlin.
(我在柏林找房子。)
```

### 教师/医生类特殊变化

```
┌─────────────────────────────────────────────────────────┐
│  阳性名词阴化规则（职业名称）：                           │
│                                                         │
│  1. 通常加 -in：                                        │

│  3. 以 -mann 结尾：                                     │
```

## 4.7 形容词变化

### 定冠词后的形容词词尾

```
┌─────────────────────────────────────────────────────────┐
│  形容词跟在定冠词后面时，需要加词尾：                      │
│                                                         │
│  阳性：-e                                               │
│  der klein-e Mann → der kleine Mann (矮个子男人)        │
│  der neu-e Wagen → der neue Wagen (新车)                │
│                                                         │
│  阴性：-e                                               │
│  die klein-e Frau → die kleine Frau (矮个子女人)        │
│  die neu-e Wohnung → die neue Wohnung (新公寓)         │
│                                                         │
│  中性：-e                                               │
│  das klein-e Kind → das kleine Kind (小孩)             │
│  das neu-e Haus → das neue Haus (新房)                  │
│                                                         │
│  复数：-en                                             │
│  die klein-en Kinder → die kleinen Kinder              │
│  die neu-en Häuser → die neuen Häuser                   │
└─────────────────────────────────────────────────────────┘
```

### 物主冠词后的形容词词尾

```
┌─────────────────────────────────────────────────────────┐
│  物主冠词 + 形容词 + 名词：                               │
│                                                         │
│  阳性：-er                                              │
│  mein klein-er Sohn                                    │
│  dein neu-er Wagen                                     │
│  sein gut-er Freund                                    │
│                                                         │
│  阴性：-e                                               │
│  meine klein-e Tochter                                 │
│  deine neu-e Wohnung                                   │
│  seine gut-e Idee                                      │
│                                                         │
│  中性：-es                                              │
│  mein klein-es Baby                                   │
│  dein neu-es Haus                                      │
│  sein gut-es Essen                                    │
│                                                         │
│  复数：-en                                             │
│  meine klein-en Kinder                                │
│  deine neu-en Häuser                                   │
│  seine gut-en Freunde                                  │
└─────────────────────────────────────────────────────────┘
```

## 4.8 形容词比较级与最高级

### 比较级

```
┌─────────────────────────────────────────────────────────┐
│  规则变化：                                              │
│  形容词 + -er                                           │
│                                                         │
│  klein → kleiner (更小)                                 │
│  groß → größer (更大)                                   │
│  jung → jünger (更年轻)                                 │
│  alt → älter (更老)                                     │
│  schön → schöner (更漂亮)                               │
│                                                         │
│  不规则变化：                                            │
│  gut → besser (更好)                                    │
│  viel → mehr (更多)                                     │
│  gern → lieber (更喜欢)                                 │
│  groß → größer (更大)                                    │
└─────────────────────────────────────────────────────────┘
```

### 最高级

```
┌─────────────────────────────────────────────────────────┐
│  规则变化：                                              │
│  形容词 + -st-(e)                                        │
│                                                         │
│  klein → kleinst- (最小)                                │
│  groß → größt- (最大)                                   │
│  schön → schönst- (最漂亮)                              │
│  klug → klügst- (最聪明)                                │
│                                                         │
│  不规则变化：                                            │
│  gut → best- (最好)                                     │
│  viel → meist- (最多)                                   │
└─────────────────────────────────────────────────────────┘
```

### 使用句型

```
┌─────────────────────────────────────────────────────────┐
│  比较：... ist + 形容词比较级 + als ...                  │
│                                                         │
│  Er ist größer als sein Bruder.                        │
│  (他比他兄弟高。)                                        │
│                                                         │
│  Ich bin jünger als meine Schwester.                    │
│  (我比我妹妹年轻。)                                      │
│                                                         │
│  ───────────────────────────────────────                │
│                                                         │
│  最高级：... ist + der/die/das + 形容词最高级            │
│                                                         │
│  Er ist der größte in unserer Klasse.                  │
│  (他是我们班最高的。)                                    │
│                                                         │
│  Das ist das beste Restaurant in der Stadt.            │
│  (这是城里最好的餐厅。)                                   │
└─────────────────────────────────────────────────────────┘
```

## 4.9 人物描述综合练习

### 描述自己的家人

```
Meine Familie:
(我的家庭：)

Mein Vater heißt Hans. Er ist 55 Jahre alt und ist 
Arzt. Er ist groß und hat kurze graue Haare. 
Er ist sehr nett und geduldig.
(我爸爸叫Hans。55岁，是医生。他高个子，短灰发。他很
友善且有耐心。)

Meine Mutter heißt Maria. Sie ist 52 Jahre alt und 
ist Lehrerin. Sie ist mittelgroß und hat lange blonde 
Haare. Sie ist sehr liebevoll und organisiert.
(我妈妈叫Maria。52岁，是老师。她中等身高，金色长发。
她很慈爱且有条理。)

Ich habe auch einen Bruder. Er heißt Peter und ist 
30 Jahre alt. Er ist der Ältere. Er ist Ingenieur 
und arbeitet bei BMW.
(我还有一个哥哥。叫Peter，30岁。他是我的哥哥。是工程
师，在宝马工作。)

Meine Schwester ist 25 Jahre alt. Sie ist die 
Jüngere. Sie studiert noch Medizin.
(我妹妹25岁。她是家中最小的。还在学医。)
```

### 描述朋友

```
Mein bester Freund:
(我最好的朋友：)

Sein Name ist Thomas. Wir kennen uns seit 10 Jahren. 
Er ist so alt wie ich. Er ist etwa 1,75 m groß und 
hat braune Haare und braune Augen.
(他叫Thomas。我们认识10年了。他和我同岁。大约1米75，
棕色头发和眼睛。)

Thomas ist ein sehr guter Freund. Er ist immer 
hilfsbereit und ehrlich. Wenn ich Probleme habe, 
kann ich immer mit ihm reden.
(Thomas是个很好的朋友。他总是乐于助人且诚实。当我有
问题时，我总是可以和他谈。)

Wir treffen uns fast jeden Wochenende. Wir gehen 
zusammen ins Kino oder spielen Fußball.
(我们几乎每个周末都见面。我们一起去看电影或踢足球。)

Er ist der beste Freund, den ich mir wünschen kann!
(他是我能想到的最好的朋友！)
```
