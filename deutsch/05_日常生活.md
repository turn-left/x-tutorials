## 5.2 日常活动动词

### 规则动词变位（现在时）

```
┌─────────────────────────────────────────────────────────┐
│  规则动词变位规律：                                       │
│                                                         │
│  词干 + 人称词尾                                         │
│                                                         │
│  以 -en 结尾的动词，去 -en 加上下列词尾：                  │
│                                                         │
│  ich      → -e                                          │
│  du       → -st                                         │
│  er/sie/es → -t                                         │
│  wir      → -en                                         │
│  ihr      → -t                                          │
│  sie/Sie  → -en                                         │
└─────────────────────────────────────────────────────────┘
```

### 以 lernen 为例

| 人称 | 变位 | 例句 |
|------|------|------|
| ich | lerne | Ich lerne Deutsch. |
| du | lernst | Du lernst schnell. |
| er/sie/es | lernt | Er lernt jeden Tag. |
| wir | lernen | Wir lernen zusammen. |
| ihr | lernt | Ihr lernt gut. |
| sie/Sie | lernen | Sie lernen fleißig. |

### 其他规则动词

| 动词原形 | ich | du | er/sie/es | 词义 |
|----------|-----|----|-----------|------|
| machen | mache | machst | macht | 做 |
| arbeiten | arbeite | arbeitest | arbeitet | 工作 |
| wohnen | wohne | wohnst | wohnt | 住 |
| spielen | spiele | spielst | spielt | 玩 |
| kochen | koche | kochst | kocht | 做饭 |
| hören | höre | hörst | hört | 听 |
| sehen | sehe | siehst | sieht | 看 |
| finden | finde | findest | findet | 找到 |

## 5.5 可分动词

### 常见可分动词

```
┌─────────────────────────────────────────────────────────┐
│  可分动词结构：                                          │
│  可分前缀 + 动词词干                                     │
│  (前缀与动词分开使用，在从句中合在一起)                   │
│                                                         │
│  aufstehen 起床                                          │
│  · Ich stehe früh auf. (我早起。)                       │
│  · Er steht um 6 Uhr auf. (他6点起床。)                  │
│                                                         │
│  其他常见可分动词：                                      │
│  · einkaufen 购物 → Ich kaufe ein.                     │
│  · fernsehen 看电视 → Ich sehe fern.                   │
│  · absagen 取消 → Ich sage ab.                         │
│  · mitkommen 一起来 → Kommst du mit?                   │
│  · anfangen 开始 → Wann fängst du an?                  │
│  · aufhören 结束/停止 → Wann hören wir auf?             │
│  · vorstellen 介绍 → Darf ich mich vorstellen?         │
│  · stattfinden 举行 → Wann findet die Party statt?      │
└─────────────────────────────────────────────────────────┘
```

### 可分动词在句中位置

```
陈述句：动词的可分前缀放在句末
┌─────────────────────────────────────────────────────────┐
│  Ich stehe früh auf.                                   │
│  (我早起。)                                              │
│                                                         │
│  Mein Freund kommt heute Abend vorbei.                 │
│  (我朋友今晚过来。)                                      │
│                                                         │
│  Wir räumen die Wohnung auf.                          │
│  (我们整理公寓。)                                        │
└─────────────────────────────────────────────────────────┘

疑问句：可分前缀与动词分开，动词提前
┌─────────────────────────────────────────────────────────┐
│  Stehst du früh auf?                                   │
│  (你早起吗？)                                           │
│                                                         │
│  Kommt er heute?                                       │
│  (他今天来吗？)                                         │
└─────────────────────────────────────────────────────────┘
```

## 5.6 频率表达

### 频率副词

| 德语 | 中文 | 频率 | 例句 |
|------|------|------|------|
| immer | 总是 | 100% | Ich bin immer pünktlich. |
| meistens | 通常 | 80% | Ich trinke meistens Kaffee. |
| oft | 经常 | 60% | Ich gehe oft ins Kino. |
| manchmal | 有时 | 40% | Ich koche manchmal selbst. |
| selten | 很少 | 20% | Ich esse selten Fleisch. |
| nie | 从不 | 0% | Ich rauche nie. |

### 频率句型

```
┌─────────────────────────────────────────────────────────┐
│  表达频率的方式：                                        │
│                                                         │
│  1. 频率副词（位于动词之前）：                            │
│     Ich trinke oft Kaffee.                              │
│     (我经常喝咖啡。)                                     │
│                                                         │
│  2. jeden Tag / jede Woche 等：                        │
│     Ich lerne jeden Tag Deutsch.                       │
│     (我每天学德语。)                                     │
│                                                         │
│  3. 一周几次：                                          │
│     Ich gehe zweimal pro Woche ins Fitnessstudio.       │
│     (我每周去两次健身房。)                               │
│                                                         │
│  4. 百分比式：                                          │
│     Ich bin zu 80% fertig.                             │
│     (我完成了80%。)                                     │
└─────────────────────────────────────────────────────────┘
```

### 频率对比

```
Immer (>90%)：
Ich trinke immer Kaffee zum Frühstück.
(我早餐总是喝咖啡。)

Meistens (60-90%)：
Ich bin meistens zu Hause.
(我通常在家。)

Oft (40-60%)：
Ich rufe dich oft an.
(我经常给你打电话。)

Manchmal (20-40%)：
Manchmal gehe ich aus.
(有时候我出去。)

Selten (<20%)：
Ich esse selten Süßigkeiten.
(我很少吃甜食。)

Nie (0%)：
Ich habe nie Hunger am Morgen.
(我早上从不饿。)
```

## 5.7 星期几与活动

### 星期活动安排

```
┌─────────────────────────────────────────────────────────┐
│  Mein Wochenplan:                                       │
│                                                         │
│  Montag:        Ich habe Deutschkurs.                 │

│  Donnerstag:    Ich gehe zum Sport.                   │

```

### 时间介词

```
┌─────────────────────────────────────────────────────────┐
│  常用时间介词：                                          │
│                                                         │
│  um + 时间点                                            │
│  · Um 8 Uhr fange ich an.                              │
│  (我8点开始。)                                          │
│                                                         │
│  am + 星期/日期                                         │
│  · Am Montag habe ich frei.                            │
│  (周一我休息。)                                          │
│                                                         │
│  um ... Uhr + am + 星期                                │
│  · Am Mittwoch um 14 Uhr habe ich Kurs.                │
│  (周三14点我有课。)                                       │
│                                                         │
│  von ... bis ...                                        │
│  · Ich arbeite von 9 bis 17 Uhr.                       │
│  (我工作从9点到17点。)                                   │
└─────────────────────────────────────────────────────────┘
```

## 5.8 描述自己的一天

### 完整示例

```
Mein Tagesablauf

Ich bin Anna und möchte euch meinen Alltag vorstellen.

Normalerweise stehe ich um halb sieben auf. Zuerst dusche
ich, dann frühstücke ich. Zum Frühstück trinke ich immer
einen Kaffee und esse Brot mit Käse.

Um Viertel nach sieben ziehe ich mich an und fahre zur
Arbeit. Ich arbeite von acht bis fünf. Mittags esse ich
in der Kantine. Die Kantine ist zwar nicht besonders
gut, aber schnell.

Nach der Arbeit treffe ich oft meine Freundin. Wir gehen
zusammen einkaufen oder trinken etwas. Manchmal kochen
wir zusammen.

Am Abend bin ich meistens müde. Dann sehe ich fern oder
lese ein Buch. Um elf Uhr gehe ich ins Bett.

Am Wochenende schlafe ich länger. Samstags putze ich die
Wohnung und wasche ich Wäsche. Sonntags koche ich immer
für meine Familie.

So sieht mein Alltag aus. Und wie sieht euer Alltag aus?
```

## 课后练习

### 练习1：动词变位

将下列动词按人称变位：

```
1. machen: ich ____, du ____, er ____, wir ____
2. sprechen: ich ____, du ____, er ____, sie ____
3. gehen: ich ____, du ____, er ____, ihr ____
4. nehmen: du ____, er ____, ich ____
5. lesen: du ____, er ____, wir ____
```

### 练习2：填入正确的动词

```
1. Ich ____ jeden Morgen Kaffee. (trinken)
2. ____ du gern? (kochen)
3. Er ____ um 7 Uhr auf. (aufstehen)
4. Wir ____ Deutsch am Abend. (lernen)
5. Sie ____ ins Kino. (gehen)
```

### 练习3：回答问题

描述你的一天（使用德语）：
- 几点起床？
- 吃什么早餐？
- 几点上班/上学？
- 中午吃什么？
- 晚上做什么？

---

*答案：*
*练习1: 1. mache, machst, macht, machen; 2. spreche, sprichst, spricht, sprechen; 3. gehe, gehst, geht, geht; 4. nimmst, nimmt, nehme; 5. liest, liest, lesen*
*练习2: 1. trinke; 2. Kochst; 3. steht; 4. lernen; 5. geht*
