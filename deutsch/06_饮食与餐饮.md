## 6.3 三餐表达

### 三餐名称

| 中文 | 德语 | 动词 | 时间 |
|------|------|------|------|
| 早餐 | das Frühstück | frühstücken | 早上 |
| 午餐 | das Mittagessen | zu Mittag essen | 中午 |
| 晚餐 | das Abendessen | zu Abend essen | 晚上 |

### 表达饮食

```
┌─────────────────────────────────────────────────────────┐
│  吃早餐/午餐/晚餐：                                      │
│  Ich frühstücke. (我吃早餐。)                           │
│  Ich esse zu Mittag. (我吃午餐。)                       │
│  Ich esse zu Abend. (我吃晚餐。)                        │
│                                                         │
│  喝早间咖啡/下午咖啡：                                   │
│  Ich trinke morgens Kaffee.                            │
│  Ich trinke nachmittags Kaffee.                        │
└─────────────────────────────────────────────────────────┘
```

### 典型德式早餐

```
Ein typisches deutsches Frühstück:

Mein Frühstück:
Ich trinke eine Tasse Kaffee.
Ich esse Brot mit Butter.
Ich esse Käse oder Wurst.
Ich esse ein Ei.
Ich esse etwas Obst.

Zum Trinken: Kaffee, Tee, Saft, Milch
Zum Essen: Brot, Brötchen, Butter, Käse, Wurst, Ei, Marmelade, Honig
```

### 典型德式午餐/晚餐

```
德式正餐通常比较丰盛：

Mittagessen (午餐):
· 汤 Suppe
· 主菜 Hauptgericht (Fleisch mit Kartoffeln oder Reis)
· 甜点 Nachtisch

Abendessen (晚餐):
德式晚餐比较简单，通常是冷餐：
· Brot mit Belag (面包配各种 toppings)
· kaltes Fleisch (冷肉)
· Käse (奶酪)
· Salat (沙拉)
```

## 6.4 饮食喜好

### 喜欢与不喜欢

```
┌─────────────────────────────────────────────────────────┐
│  mögen (喜欢) 的变位：                                   │
│  ich mag, du magst, er/sie mag, wir mögen, ihr mögt   │
│                                                         │
│  gern haben / gern essen / gern trinken：              │
│  Ich esse gern Fisch. (我喜欢吃鱼。)                    │
│  Ich trinke gern Kaffee. (我喜欢喝咖啡。)               │
│  Ich mag Schokolade. (我喜欢巧克力。)                   │
│                                                         │
│  不喜欢：nicht gern / nicht mögen                       │
│  Ich esse nicht gern Gemüse. (我不喜欢吃蔬菜。)         │
│  Ich mag keinen Kaffee. (我不喜欢咖啡。)                 │
└─────────────────────────────────────────────────────────┘
```

### 食物评价

```
Das schmeckt gut.         (好吃。)
Das schmeckt lecker.      (很美味。)
Das schmeckt nicht.       (不好吃。)
Das schmeckt bitter.      (苦的。)
Das schmeckt süß.         (甜的。)
Das schmeckt sauer.       (酸的。)
Das schmeckt scharf.      (辣的。)

Ich habe Hunger.          (我饿了。)
Ich habe Durst.           (我渴了。)
Ich bin satt.            (我饱了。)
```

### 对话示例

```
A: Was möchtest du essen?
   (你想吃什么？)

B: Ich möchte Pizza bestellen.
   (我想点披萨。)

A: Möchtest du auch etwas trinken?
   (你想喝点什么吗？)

B: Ja, ich möchte eine Cola, bitte.
   (好的，请给我一杯可乐。)

A: Magst du auch einen Salat?
   (你也喜欢沙拉吗？)

B: Nein danke, ich mag keinen Salat.
   (不用了谢谢，我不喜欢沙拉。)
```

## 6.5 在餐厅

### 进入餐厅

```
Einen Tisch für zwei Personen, bitte.
(请给我们一张两人桌。)

Haben Sie einen Tisch reserviert?
(您预约了吗？)

Ist dieser Platz frei?
(这个位置空吗？)

Wir möchten bestellen.
(我们想点餐。)
```

### 询问菜单

```
Die Speisekarte, bitte.
(请给我菜单。)

Was empfehlen Sie?
(您推荐什么？)

Was gibt es heute?
(今天有什么？)

Gibt es ein Tagesgericht / Empfehlung?
(有今日特餐/推荐吗？)

Ich bin Vegetarier / Veganer.
(我是素食者/纯素食者。)

Haben Sie auch Kindergerichte?
(你们有儿童餐吗？)
```

### 点餐表达

```
Ich möchte bestellen.
(我想点餐。)

Ich hätte gern...
(我想点...)

Ich nehme...
(我点...)

Für mich bitte...
(请给我...)

Als Vorspeise / Hauptgericht / Nachtisch...
(作为前菜/主菜/甜点...)

Die Portion ist sehr groß/klein.
(份量很大/小。)

Noch ein Bier, bitte.
(再来一杯啤酒，谢谢。)

Die Rechnung, bitte.
(请买单。)

Zahlen, bitte.
(买单。)

Ich möchte bezahlen.
(我想付款。)

Möchten Sie mit Karte oder bar zahlen?
(您想刷卡还是付现金？)

Kann ich mit Kreditkarte zahlen?
(我可以刷信用卡吗？)

Wir möchten getrennt zahlen.
(我们想分开付。)

Das stimmt so.
(不用找了。)

Lassen Sie das Restgeld.
(不用找钱了。)
```

### 完整餐厅对话

```
Kellner: Guten Abend! Willkommen im Restaurant!
         Haben Sie einen Tisch reserviert?

Gast:    Ja, auf den Namen Müller.

Kellner: Ja, natürlich. Folgen Sie mir bitte.
         Hier ist Ihr Tisch.

Gast:    Danke schön. Die Speisekarte, bitte.

Kellner: Bitte schön. Möchten Sie schon etwas trinken?

Gast:    Ja, ein Glas Weißwein, bitte.
         Und ein Wasser ohne Kohlensäure.

Kellner: Sehr gern. Möchten Sie auch schon bestellen
         oder brauchen Sie noch etwas Zeit?

Gast:    Ich möchte die Suppe als Vorspeise.
         Und dann den Schweinebraten als Hauptgericht.

Kellner: Und was möchten Sie trinken zum Essen?

Gast:    Ein Bier, bitte.

Kellner: Sehr gern. Die Suppe, den Schweinebraten
         und ein Bier. Kommt sofort!

... (一段时间后) ...

Kellner: So, hier ist Ihre Suppe. Und der
         Schweinebraten. Guten Appetit!

Gast:    Danke schön!

... (用餐结束后) ...

Gast:    Herr Ober! Die Rechnung, bitte.

Kellner: Ja, natürlich. Hier bitte.

Gast:    Ich möchte mit Karte zahlen.

Kellner: Kein Problem. Möchten Sie quittieren?

Gast:    Ja, hier. Vielen Dank. Das Essen war
         wirklich lecker!

Kellner: Freut mich sehr. Auf Wiedersehen und
         gute Fahrt!
```

## 6.6 第四格（第四宾语）

### 第四格概念

```
┌─────────────────────────────────────────────────────────┐
│  什么是第四格？                                          │
│                                                         │
│  德语名词有四个"格"（Kasus），表示名词在句子中的功能     │
│                                                         │
│  第一格（Nominativ）：主语                               │
│  Ich trinke Kaffee. (我喝咖啡。)                        │
│           ↑主语                                          │
│                                                         │
│  第四格（Akkusativ）：直接宾语                           │
│  Ich trinke Kaffee. (我喝咖啡。)                        │
│                  ↑宾语（第四格）                         │
│                                                         │
│  当动词需要一个宾语时，这个宾语通常用第四格               │
└─────────────────────────────────────────────────────────┘
```

### 定冠词的第四格变化

|  | 阳性 | 阴性 | 中性 | 复数 |
|--|------|------|------|------|
| 第一格 | der | die | das | die |
| 第四格 | den | die | das | die |

```
┌─────────────────────────────────────────────────────────┐
│  阳性名词的定冠词从 der 变成 den：                       │
│                                                         │
│  Ich sehe den Mann. (我看见那个男人。)                   │
│  Ich kaufe das Buch. (我买书。) → 不变                  │
│  Ich kenne die Frau. (我认识那个女人。) → 不变          │
│  Ich liebe die Kinder. (我爱孩子们。) → 不变           │
│                                                         │
│  记忆技巧：阳性名词"变变变"，其他都不变                   │
│  (除了阳性特殊变化外，阴性、中性、复数都没变化)            │
└─────────────────────────────────────────────────────────┘
```

### 不定冠词的第四格变化

|  | 阳性 | 阴性 | 中性 |
|--|------|------|------|
| 第一格 | ein | eine | ein |
| 第四格 | einen | eine | ein |

```
阳性名词的 ein → einen：
Ich habe einen Hund. (我有一只狗。)

阴性 ein → eine（不变）：
Ich habe eine Katze. (我有一只猫。)

中性 ein → ein（不变）：
Ich habe ein Auto. (我有一辆车。)
```

### 否定：kein/keine 的第四格

|  | 阳性 | 阴性 | 中性 | 复数 |
|--|------|------|------|------|
| 第一格 | kein | keine | kein | keine |
| 第四格 | keinen | keine | kein | keine |

```
┌─────────────────────────────────────────────────────────┐
│  否定：ein → kein                                      │
│                                                         │
│  Ich habe ein Auto. → Ich habe kein Auto.              │
│  (我有一辆车。)       (我没有车。)                       │
│                                                         │
│  Ich habe eine Katze. → Ich habe keine Katze.          │
│  (我有一只猫。)        (我没有猫。)                       │
│                                                         │
│  注意：复数否定都用 keine                               │
│  Ich habe Kinder. → Ich habe keine Kinder.            │
│  (我有孩子。)           (我没有孩子。)                    │
└─────────────────────────────────────────────────────────┘
```

### 物主冠词的第四格

|  | 阳性 | 阴性 | 中性 | 复数 |
|--|------|------|------|------|
| mein | meinen | meine | mein | meine |
| dein | deinen | deine | dein | deine |
| sein | seinen | seine | sein | seine |
| ihr | ihren | ihre | ihr | ihre |

## 6.7 情态动词 möchte

### möchte 的变位

```
┌─────────────────────────────────────────────────────────┐
│  möchte 是 "想要"，是 mögen 的委婉形式                   │
│                                                         │
│  ich möchte                                            │
│  du möchtest                                           │
│  er/sie/es möchte                                      │
│  wir möchten                                           │
│  ihr möchtet                                           │
│  sie/Sie möchten                                       │
└─────────────────────────────────────────────────────────┘
```

### möchte 的用法

```
┌─────────────────────────────────────────────────────────┐
│  möchte 的使用场景：                                    │
│                                                         │
│  1. 表达想要某物：                                      │
│     Ich möchte einen Kaffee. (我想喝杯咖啡。)            │
│     Ich möchte ein Bier. (我想喝杯啤酒。)               │
│                                                         │
│  2. 表达想要做某事：                                    │
│     Ich möchte Deutsch lernen. (我想学德语。)          │
│     Ich möchte nach Deutschland fahren.                 │
│     (我想去德国。)                                       │
│                                                         │
│  3. 礼貌请求：                                          │
│     Ich möchte bitte die Speisekarte.                  │
│     (请给我菜单。)                                       │
│                                                         │
│  4. 委婉表达想要：                                      │
│     Möchten Sie etwas trinken? (您想喝点什么？)         │
│     Möchtest du kommen? (你想来吗？)                   │
└─────────────────────────────────────────────────────────┘
```

### möchte vs. wollen

```
┌─────────────────────────────────────────────────────────┐
│  möchte vs. wollen：                                   │
│                                                         │
│  möchte - 委婉，想要（但不一定能实现）                   │
│  Ich möchte ein Eis, aber es gibt keins.               │
│  (我想吃冰淇淋，但是没有了。)                              │
│                                                         │
│  wollen - 强烈意愿，一定要（表示决心）                   │
│  Ich will nach Deutschland. (我一定要去德国。)         │
│  (表达强烈愿望或决定)                                    │
└─────────────────────────────────────────────────────────┘
```

## 6.8 在超市/商店买食物

### 购买表达

```
Ich möchte 500 Gramm Käse.
(我要500克奶酪。)

Ich hätte gern ein Kilo Äpfel.
(我想买一公斤苹果。)

Geben Sie mir bitte 200 Gramm Schinken.
(请给我200克火腿。)

Was kostet das?
(这个多少钱？)

Ist das bio / regional / frisch?
(这是有机的/本地的/新鲜的吗？)

Haben Sie auch Vollkornbrot?
(你们也有全麦面包吗？)

Ich nehme das.
(我要这个。)

Das ist alles, danke.
(就这些，谢谢。)

Kann ich eine Tüte haben?
(能给我一个袋子吗？)
```

### 练习3：餐厅情景对话

用所给词组织对话：
bestellen, möchte, die Rechnung, zahlen, bitte

```
Kellner: Guten Abend! Was _____ Sie?

Gast:    Ich _____ die Suppe.

Kellner: Und zu trinken?

Gast:    Einen Wein, _____.

Kellner: Möchten Sie auch -----?

Gast:    Nein danke, das ist alles.

Kellner: Hier bitte. 

Gast:    Ich möchte _____, bitte. Mit Karte.

Kellner: Kein Problem.
```

---

*答案：*
*练习1: 1. den Mann; 2. den Kaffee; 3. das Brot; 4. die Katze; 5. die Kinder*
*练习2: 1. habe kein; 2. trinkt keine; 3. hat keine; 4. essen keine*
*练习3: möchten; möchte; bitte; bestellen; die Rechnung; zahlen*
