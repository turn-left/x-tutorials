### 试穿相关表达

```

(这个有其他尺码吗？)

(这个有38码的吗？)

Haben Sie das in kleiner/größer?
(这个有小一号/大一号的吗？)

Haben Sie das in anderen Farben?
(这个有其他颜色的吗？)

Kann ich es anprobieren?
(我可以试穿吗？)

Wo ist die Umkleidekabine / der Probierraum?
(试衣间在哪里？)

Es passt mir nicht.
(不合身。)

Es ist zu groß / zu klein.
(太大/太小。)

Es ist zu eng / zu weit.
(太紧/太松。)

Es steht mir gut.
(很适合我。)

Ich nehme es.
(我买了。)
```

## 7.4 购物对话

### 在服装店

```
Verkäufer: Guten Tag! Kann ich Ihnen helfen?
           (您好！需要帮忙吗？)

Kunde:     Ja, ich suche ein Kleid für eine Party.
           (是的，我在找一条参加聚会的裙子。)

Verkäufer: Was für ein Kleid möchten Sie denn?
           (您想要什么样的裙子？)

Kunde:     Etwas Elegantes, vielleicht schwarz oder rot.
           (优雅一点的，可能是黑色或红色。)

Verkäufer: Wir haben hier mehrere Modelle. Welche Farbe
           gefällt Ihnen besser?
           (我们这里有几款。您更喜欢哪个颜色？)

Kunde:     Das schwarze bitte. Und ich möchte es
           anprobieren.
           (请给我黑色的。我想试穿一下。)

Verkäufer: Natürlich. Die Umkleidekabine ist dort links.
           (当然。试衣间在那边左边。)

Kunde:     (nach dem Anprobieren)
           (试穿后)
           Wie steht es mir?
           (我穿起来怎么样？)

Verkäufer: Es steht Ihnen sehr gut! Passt perfekt.
           (非常适合您！非常合身。)

Kunde:     Super! Ich nehme es. Haben Sie die passenden
           Schuhe dazu?
           (太好了！我买了。有配套的鞋子吗？)

Verkäufer: Ja, wir haben schwarze High Heels.
           (有的，我们有黑色高跟鞋。)

Kunde:     Kann ich die auch anprobieren?
           (我可以试穿一下吗？)

Verkäufer: Aber natürlich. Welche Größe haben Sie?
           (当然可以。您穿多大码？)

Kunde:     Größe 38.
           (38码。)

Verkäufer: Bitte sehr.
           (请。)
```

## 7.5 价格与付款

### 价格表达

```
┌─────────────────────────────────────────────────────────┐
│  价格表达方式：                                          │
│                                                         │
│  Das kostet 10 Euro. (这个10欧元。)                     │
│  Der Preis ist 10 Euro. (价格是10欧元。)                │
│  Es kostet nur 5 Euro. (只要5欧元。)                    │
│  Es kostet etwas / viel / wenig.                        │
│  (有点贵/很贵/不贵。)                                    │
│                                                         │
│  找零钱：                                              │
│  Kann ich wechseln? (能换零钱吗？)                      │
│  Haben Sie 5 Euro? (您有5欧元吗？)                      │

```

| 付款方式 | 德语 | 备注 |

| 现金 | bar / das Bargeld | 德国仍常用现金 |

| 借记卡 | die EC-Karte / die Debitkarte | 德国常用 |
| 手机支付 | das Mobiltelefon / Apple Pay / Google Pay | 越来越流行 |
| 转账 | die Überweisung | 银行转账 |

```

(我刷卡。)

(我手机支付。)

Nehmen Sie Kreditkarten?
(你们接受信用卡吗？)
```

## 7.7 情态动词 können

### können 的变位

```
┌─────────────────────────────────────────────────────────┐
│  können（能、会）的变位：                                │
│                                                         │
│  ich kann      du kannst     er/sie/es kann            │
│  wir können    ihr könnt     sie/Sie können            │
│                                                         │
│  过去时：konnte                                          │
│  ich konnte    du konntest   er/sie/es konnte          │
└─────────────────────────────────────────────────────────┘
```

### können 的用法

```
┌─────────────────────────────────────────────────────────┐
│  können 的使用场景：                                    │
│                                                         │
│  1. 表示能力"会/能"：                                   │
│     Ich kann Deutsch sprechen.                         │
│     (我会说德语。)                                      │
│     Ich kann nicht schwimmen.                          │
│     (我不会游泳。)                                       │
│                                                         │
│  2. 表示可能性"可能"：                                  │
│     Das kann sein.                                     │
│     (这有可能。)                                        │
│     Das kann ich nicht machen.                         │
│     (这我做不到。)                                      │
│                                                         │
│  3. 表示许可"可以"：                                    │
│     Kann ich hier rauchen?                             │
│     (我可以在这里吸烟吗？)                               │
│     Kann ich die Karte haben?                          │
│     (可以给我看看菜单吗？)                               │
└─────────────────────────────────────────────────────────┘
```

### können vs. möchte

```
┌─────────────────────────────────────────────────────────┐
│  können vs. möchte：                                   │
│                                                         │
│  können - 能力/可能性                                   │
│  Ich kann schwimmen. (我会游泳。) → 能力                │
│  Ich kann das machen. (我能做这个。) → 可能性           │
│                                                         │
│  möchte - 愿望/请求                                     │
│  Ich möchte ein Eis. (我想吃冰淇淋。) → 愿望            │
│  Ich möchte zahlen. (我想买单。) → 请求                 │
│                                                         │
│  Ich kann das nicht. (我做不到。)                      │
│  Ich möchte das nicht. (我不想做这个。)                 │
│  两者意思不同！                                          │
└─────────────────────────────────────────────────────────┘
```

### 网购对话

```
A: Ich möchte dieses Kleid online bestellen.
   (我想网购这条裙子。)

B: Welche Größe haben Sie?
   (您穿什么尺码？)

A: Ich habe Größe M.
   (我穿M码。)

B: Möchten Sie es nach Hause liefern lassen?
   (您想送货到家吗？)

A: Ja, bitte. Wie viel kostet die Lieferung?
   (好的。运费多少钱？)

B: Die Lieferung ist kostenlos ab 50 Euro.
   (满50欧免运费。)

A: Wie lange dauert die Lieferung?
   (送货要多久？)

B: Ungefähr 3-5 Werktage.
   (大约3-5个工作日。)

A: Kann ich es zurückgeben, wenn es nicht passt?
   (如果不合身可以退货吗？)

B: Ja, Sie haben 14 Tage Widerrufsrecht.
   (可以的，您有14天退货期。)

A: Perfekt. Dann bestelle ich es.
   (好的。那我下单了。)
```
