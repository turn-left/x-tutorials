## 8.1 出行方式

### 常见出行方式

| 中文 | 德语 | 例句 |
|------|------|------|
| 步行 | zu Fuß gehen | Ich gehe zu Fuß. |
| 骑自行车 | Fahrrad fahren | Ich fahre Fahrrad. |
| 坐公交车 | Bus fahren | Ich fahre Bus. |
| 坐地铁 | U-Bahn fahren | Ich fahre U-Bahn. |
| 坐火车 | Zug fahren | Ich fahre Zug. |
| 坐船 | Schiff fahren | Ich fahre Schiff. |
| 坐飞机 | fliegen | Ich fliege. |
| 开车 | Auto fahren | Ich fahre Auto. |
| 打车 | Taxi fahren | Ich nehme ein Taxi. |
| 搭便车 | per Anhalter fahren | - |

### 交通方式详细用法

```
┌─────────────────────────────────────────────────────────┐
│  mit + 交通工具：                                       │
│                                                         │
│  mit dem Auto fahren    (开车)                         │
│  mit dem Bus fahren     (坐公交车)                     │
│  mit der U-Bahn fahren  (坐地铁)                       │
│  mit dem Zug fahren     (坐火车)                       │
│  mit dem Fahrrad fahren (骑自行车)                     │
│  mit dem Taxi fahren    (打车)                         │
│  mit dem Flugzeug fliegen (坐飞机)                     │
│  mit dem Schiff fahren  (坐船)                         │
│  zu Fuß gehen           (步行 - 不用mit)               │
│                                                         │
│  例句：                                                 │
│  Ich fahre mit dem Bus zur Arbeit.                    │
│  (我坐公交上班。)                                       │
│                                                         │
│  Ich gehe zu Fuß zur Schule.                          │
│  (我步行上学。)                                         │
└─────────────────────────────────────────────────────────┘
```

### 动词 fahren 的变位

| 人称 | 变位 | 例句 |
|------|------|------|
| ich | fahre | Ich fahre nach Berlin. |
| du | fährst | Du fährst sehr schnell. |
| er/sie/es | fährt | Er fährt mit dem Zug. |
| wir | fahren | Wir fahren in den Urlaub. |
| ihr | fahrt | Fahrt ihr auch? |
| sie/Sie | fahren | Sie fahren nach Hause. |

### 其他方向动词

| 动词 | ich | du | er/sie/es | 词义 |
|------|------|----|-----------|------|
| gehen | gehe | gehst | geht | 走/去 |
| fliegen | fliege | fliegst | fliegt | 飞 |
| laufen | laufe | läufst | läuft | 跑 |
| reisen | reise | reist | reist | 旅行 |

## 8.2 方位词

### 基本方位副词

| 中文 | 德语 | 反义词 |
|------|------|--------|
| 上 | oben | unten |
| 下 | unten | oben |

| 右 | rechts | links |

| 后 | hinten | vorne |

| 外面 | außen / draußen | innen / drinnen |

| 对面 | gegenüber | - |

```
┌─────────────────────────────────────────────────────────┐
│  方位介词（回答 Wo? - 在哪里）：                         │
│                                                         │
│  in + 第三格                                           │

│  in der Stadt (在城市里)                               │

│                                                         │

│  auf dem Markt (在市场上)                              │
│                                                         │
│  an + 第三格                                           │
│  am Fenster (在窗户旁)                                 │
│  am Meer (在海边)                                      │
│  an der Wand (在墙上)                                  │
│                                                         │
│  vor + 第三格 (在...前面)                              │
│  vor dem Haus (在房子前面)                             │
│                                                         │
│  hinter + 第三格 (在...后面)                            │
│  hinter der Schule (在学校后面)                        │
│                                                         │

│                                                         │

│  zwischen den Bäumen (在树之间)                        │

```

### 方向介词（回答 Wohin? - 去哪里）

```
┌─────────────────────────────────────────────────────────┐
│  方向介词（回答 Wohin? - 去哪里）+ 第四格：              │
│                                                         │
│  in + 第四格                                           │
│  in die Schule gehen (去学校)                          │
│  in die Stadt fahren (去城市)                          │
│  ins Kino gehen (去看电影)                             │
│                                                         │
│  auf + 第四格                                           │
│  auf die Straße gehen (上街)                           │
│  auf den Markt gehen (去市场)                          │
│                                                         │
│  an + 第四格                                           │
│  an die Wand hängen (挂到墙上)                         │
│  ans Meer fahren (去海边)                              │
│                                                         │
│  vor + 第四格 (到...前面)                              │
│  vor das Haus gehen (走到房子前面)                     │
│                                                         │
│  hinter + 第四格 (到...后面)                            │
│  hinter das Haus gehen (走到房子后面)                   │
│                                                         │
│  neben + 第四格 (到...旁边)                            │
│  neben mich setzen (坐到我旁边)                         │
│                                                         │
│  zwischen + 第四格 (到...之间)                          │
│  zwischen die Bäume stellen (放到树之间)               │
└─────────────────────────────────────────────────────────┘
```

### 静三动四规则

```
┌─────────────────────────────────────────────────────────┐
│  德语介词的两面性：                                      │
│                                                         │
│  德语介词根据动词方向决定用第三格还是第四格：             │
│                                                         │
│  Wo? (在哪里) → 第三格（静态）                          │
│  Wohin? (去哪里) → 第四格（动态）                      │
│                                                         │
│  示例：                                                  │
│  ┌─────────────────────────────────────────────────┐   │
│  │ Ich sitze auf dem Stuhl. (坐在椅子上)           │   │
│  │ Ich setze mich auf den Stuhl. (坐到椅子上)      │   │
│  │                                    ↑第三格 ↑第四格 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ Das Bild hängt an der Wand. (挂在墙上)          │   │
│  │ Ich hänge das Bild an die Wand. (挂到墙上)      │   │
│  │                                    ↑第三格 ↑第四格 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  可以"静三动四"的介词（9个）：                           │
│  in, auf, an, vor, hinter, neben, zwischen, unter, über │
└─────────────────────────────────────────────────────────┘
```

## 8.3 问路表达

### 询问方向

```
Entschuldigung, wo ist...?
(打扰一下，...在哪里？)

Entschuldigung, wie komme ich zu...?
(打扰一下，我怎么去...？)

Entschuldigung, kann ich hier zu Fuß zu...?
(打扰一下，这里步行能到...吗？)

Ist es weit von hier?
(离这里远吗？)

Ist es in der Nähe?
(在附近吗？)

Kann man von hier zu Fuß hingehen?
(能步行过去吗？)

Wie lange dauert es zu Fuß / mit dem Bus?
(步行/坐公交要多久？)

Welche Buslinie fährt dahin?
(哪路公交去那里？)

Wo ist der nächste Bahnhof?
(最近的火车站在哪里？)

Können Sie mir den Weg zeigen?
(您能给我指路吗？)
```

### 完整指路对话

```
A: Entschuldigung, wie komme ich zum Hauptbahnhof?
   (打扰一下，我怎么去中央火车站？)

B: Gehen Sie hier geradeaus bis zur Ampel.
   (从这里直走，到红绿灯处。)

A: Und dann?
   (然后呢？)

B: Dann biegen Sie rechts ab.
   (然后右转。)

A: Rechts ab. Und dann?
   (右转。然后呢？)

B: Gehen Sie weiter bis zur nächsten Kreuzung.
   (继续走到下一个十字路口。)

A: Ist der Bahnhof dann links oder rechts?
   (那火车站在左边还是右边？)

B: Der Bahnhof ist auf der linken Seite, direkt neben
   dem Hotel "Berliner Hof".
   (火车站在左边，就在"柏林人酒店"旁边。)

A: Vielen Dank!
   (非常感谢！)

B: Kein Problem! Gute Fahrt!
   (不客气！一路顺风！)

---

A: Entschuldigung, ist der Supermarkt hier in der Nähe?
   (打扰一下，超市在附近吗？)

B: Ja, der nächste Supermarkt ist nicht weit.
   (是的，最近的超市不远。)

A: Wie komme ich dorthin?
   (我怎么去那里？)

B: Gehen Sie die Straße entlang.
   (沿着这条街走。)

A: Die Straße entlang. Und dann?
   (沿着街走。然后呢？)

B: An der zweiten Kreuzung links.
   (在第二个十字路口左转。)

A: An der zweiten Kreuzung links. Gibt es einen Parkplatz?
   (第二个十字路口左转。有停车场吗？)

B: Ja, hinter dem Supermarkt ist ein Parkplatz.
   (有的，超市后面有停车场。)

A: Danke schön!
   (非常感谢！)
```

### 详细行程描述

```
Meine Reise nach München:

Ich fahre morgen früh von zu Hause ab. Zuerst fahre ich
mit dem Bus zum Bahnhof. Dort nehme ich den ICE nach
München. Die Fahrt dauert ungefähr 6 Stunden.

Wenn ich in München ankomme, nehme ich die U-Bahn zum
Hotel. Das Hotel ist in der Stadtmitte, neben dem Marienplatz.

Am Abend gehe ich zu Fuß in ein Restaurant. Das Restaurant
ist nicht weit vom Hotel.

Am nächsten Tag möchte ich mit dem Fahrrad durch die Stadt
fahren. Man kann nämlich Fahrräder mieten.
```

## 8.7 Wohin? Wo? 区分练习

### 静三动四对比

```
┌─────────────────────────────────────────────────────────┐
│  Wo? (在哪里) - 用第三格                                 │
│                                                         │
│  Das Buch liegt auf dem Tisch.                         │
│  (书在桌子上。)                                          │
│                                                         │
│  Ich wohne in der Stadt.                               │
│  (我住在城市里。)                                        │
│                                                         │
│  Der Schirm steht neben der Tür.                        │
│  (伞在门旁边。)                                          │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  Wohin? (去哪里) - 用第四格                              │
│                                                         │
│  Ich lege das Buch auf den Tisch.                      │
│  (我把书放到桌子上。)                                     │
│                                                         │
│  Ich ziehe um. Ich ziehe in die Stadt.                 │
│  (我搬家。我搬进城市里。)                                 │
│                                                         │
│  Ich stelle den Schirm neben die Tür.                   │
│  (我把伞放到门旁边。)                                     │
└─────────────────────────────────────────────────────────┘
```

### 更多例句对比

```

─────────────────────────────────────────────────────

(坐在椅子上)           (坐到椅子上)

(在学校)               (去学校)

im Kino sein          ins Kino gehen
(在电影院)             (去看电影)

(住在海边)             (去海边)
```

### 练习1：方位词填空

```
1. Der Bahnhof ist ____ dem Hotel. (在旁边)

4. Die Katze ist ____ dem Tisch. (在下面)

```

### 练习2：静三动四选择

用合适的介词填空（注意是用第三格还是第四格）：

```
1. Ich lege das Buch auf ___ Tisch. (Wo? auf dem / Wohin? auf den)

2. Das Bild hängt ___ Wand. (Wo? an der / Wohin? an die)

3. Ich setze mich ___ Stuhl. (Wo? auf dem / Wohin? auf den)

4. Wir wohnen ___ Stadt. (Wo? in der / Wohin? in die)

5. Der Hund läuft ___ Haus. (Wo? hinter dem / Wohin? hinter das)
```

### 练习3：描述路线

描述从你家到最近超市的路线，用德语写出：
- 出发点和目的地
- 行走方向（左/右/直走）
-标志性建筑
- 需要的时间

---

*答案：*
*练习1: 1. neben; 2. gegenüber; 3. vor; 4. unter; 5. zwischen*
*练习2: 1. auf den; 2. an der; 3. auf den; 4. in der; 5. hinter das*
