## 1. 德语词性

### 三种词性

| 词性 | 定冠词 | 不定冠词 | 例子 |
|:----:|:------:|:--------:|------|
| 阳性 | der | ein | der Mann, ein Vater |
| 阴性 | die | eine | die Frau, eine Mutter |
| 中性 | das | ein | das Kind, ein Baby |

### 词性记忆规律

```
┌─────────────────────────────────────────────────────────┐
│  阳性特征（der）：                                       │
│  · 男性：der Mann, der Vater                          │
│  · 以 -er 结尾：der Lehrer, der Fischer               │
│  · 方向、风、季节、月份：der Wind, der Sommer         │
│                                                         │
│  阴性特征（die）：                                       │
│  · 女性：die Frau, die Mutter                        │
│  · 以 -e 结尾：die Lampe, die Tasse                  │
│  · 以 -ung, -heit, -keit 结尾：die Wohnung, die Freiheit │
│                                                         │
│  中性特征（das）：                                       │
│  · 以 -chen, -lein 结尾：das Mädchen, das Büchlein   │
│  · 动词不定式名词化：das Essen, das Trinken            │
│  · 国名、语言：das Deutsch, das China                 │
└─────────────────────────────────────────────────────────┘
```

## 2. 定冠词与不定冠词

### 定冠词

|  | 阳性 | 阴性 | 中性 | 复数 |
|:--:|:----:|:----:|:----:|:----:|
| 第一格 | der | die | das | die |
| 第四格 | den | die | das | die |

### 不定冠词

|  | 阳性 | 阴性 | 中性 |
|:--:|:----:|:----:|:----:|
| 第一格 | ein | eine | ein |
| 第四格 | einen | eine | ein |

### 使用场景

```
定冠词 der/die/das 使用：
· 再次提到的事物
· 独一无二的事物（太阳、月亮）
· 整体概念
· 最高级

不定冠词 ein/eine/ein 使用：
· 首次提到的事物
· 职业
· 表示"一个/某个"
```

## 3. 名词变格（第一格与第四格）

### 定冠词变化

|  | 阳性 | 阴性 | 中性 | 复数 |
|:--:|:----:|:----:|:----:|:----:|
| 第一格 | der | die | das | die |
| 第四格 | **den** | die | das | die |

### 不定冠词变化

|  | 阳性 | 阴性 | 中性 |
|:--:|:----:|:----:|:----:|
| 第一格 | ein | eine | ein |
| 第四格 | **einen** | eine | ein |

### 物主冠词变化（mein 为例）

|  | 阳性 | 阴性 | 中性 | 复数 |
|:--:|:----:|:----:|:----:|:----:|
| 第一格 | mein | meine | mein | meine |
| 第四格 | **meinen** | meine | mein | meine |

### 重要规律

```
┌─────────────────────────────────────────────────────────┐
│  阳性名词在第四格：                                      │
│  · 定冠词 der → den                                     │
│  · 不定冠词 ein → einen                                │
│  · 物主冠词 mein → meinen                              │
│                                                         │
│  其他词性（阴性、中性、复数）第四格不变！                  │
│                                                         │
│  记忆口诀：阳性"变变变"，其他"不变"                       │
└─────────────────────────────────────────────────────────┘
```

## 5. 物主冠词

### 变格表

| 人称 | 阳性 | 阴性 | 中性 | 复数 |
|:----:|:----:|:----:|:----:|:----:|
| ich | mein | meine | mein | meine |
| du | dein | deine | dein | deine |
| er | sein | seine | sein | seine |
| sie | ihr | ihre | ihr | ihre |
| es | sein | seine | sein | seine |
| wir | unser | unsere | unser | unsere |
| ihr | euer | eure | euer | eure |
| sie/Sie | ihr | ihre | ihr | ihre |

### 使用示例

```
Das ist mein Haus. (这是我的房子。)
Das ist deine Katze. (这是你的猫。)
Das ist sein Buch. (这是他的书。)
Das ist ihre Tasche. (这是她的包。)
Das ist unser Auto. (这是我们的车。)
```

## 9. 否定

### kein/e/kein 的使用

|  | 阳性 | 阴性 | 中性 | 复数 |
|:--:|:----:|:----:|:----:|:----:|
| 第一格 | kein | keine | kein | keine |
| 第四格 | **keinen** | keine | kein | keine |

### 用法

```
Ich habe ein Auto. → Ich habe kein Auto.
(我有一辆车。)       (我没有车。)

Ich habe eine Katze. → Ich habe keine Katze.
(我有一只猫。)        (我没有猫。)

Ich habe Kinder. → Ich habe keine Kinder.
(我有孩子。)          (我没有孩子。)
```

### nicht 的使用

```
┌─────────────────────────────────────────────────────────┐
│  nicht 用于否定：                                       │
│                                                         │
│  1. 否定整个句子：                                     │
│     Ich komme nicht. (我不来。)                        │
│                                                         │
│  2. 否定动词（位于句末或动词前）：                       │
│     Das ist nicht gut. (这个不好。)                    │
│                                                         │
│  3. 否定介词短语：                                     │
│     Ich gehe nicht zur Schule. (我不去学校。)          │
│                                                         │
│  4. 否定形容词/副词：                                  │
│     Er ist nicht krank. (他没生病。)                   │
└─────────────────────────────────────────────────────────┘
```

### 否定词

| 否定词 | 中文 | 例句 |
|--------|------|------|
| nicht | 不 | Ich weiß nicht. (我不知道。) |
| kein | 没有 | Ich habe kein Geld. (我没钱。) |
| nie/niemals | 从不 | Ich rauche nie. (我从不吸烟。) |
| nichts | 没什么 | Ich mache nichts. (我什么都没做。) |
| niemand | 没人 | Niemand kommt. (没人来。) |
| nowhere | 不...任何地方 | Ich gehe nirgendwohin. (我哪儿都不去。) |

## 10. 特殊疑问词

| wer | 谁 | Wer bist du? (你是谁？) |
